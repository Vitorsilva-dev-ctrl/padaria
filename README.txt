Padaria do Felipe - Site Oficial
===============================

Descrição:
----------
Site institucional da Padaria do Felipe, apresentando os produtos, história, contato e localização.

Como usar:
----------
- Abra o arquivo index.html no navegador para visualizar o site.
- As imagens estão organizadas na pasta "assets/icons".
